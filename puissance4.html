Puissance 4

Réalisation d'un Puissance 4 en Langage C dans le cadre d'un projet universitaire. Ce projet a été réalisé pour être affiché dans un terminal Linux, et être utilisé par des enfants. J'ai réalisé pour ce projet les différentes étapes, allant du maquettage de l'affichage souhaité pour les différents cas dans le terminal jusqu'au développement du programme, puis l'élaboration d'un cahier de tests. Cela m'a ainsi permis de découvrir les différentes étapes auquel un développeur pourrait être confronté suite à une demande de développement d'un client.

Compétences acquises :

- Autonomie - Implémentation - Tests
