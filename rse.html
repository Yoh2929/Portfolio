Compétence 6 : COLLABORER, travailler dans une équipe informatique

Réalisation d'une étude RSE

Ce projet nous as amenés à réaliser, par groupes de 4, une étude RSE sur une entreprise de notre choix. Nous avons donc choisi le groupe LVMH, car nous savions que nous allons trouver des données sur les politiques RSE, étant donné qu'il s'agit du premier groupe mondial de luxe. Nous avons été amenés dans ce projet à réaliser une matrice SWOT, le diagnostic externe de l'organisation. Ce projet nous as permis d'apprendre à analyser l'environnement d'une entreprise, de nous améliorer à l'oral

Apprentissages critiques du projet :

- AC16.01 : Appréhender l’écosystème numérique
- AC16.02 : Découvrir les aptitudes requises selon les différents secteurs informatiques
- AC16.03 : Identifier les statuts, les fonctions et les rôles de chaque membre d’une équipe pluridisciplinaire
- AC16.04 : Acquérir les compétences interpersonnelles pour travailler en équipe

Compétences acquises :

- Synthèse
- Recherche
- Analyse
